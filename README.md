今天(19日)雲林地區受低壓帶及西南風增強影響，為陰短暫陣雨或雷雨的天氣。氣溫方面：沿海區氣溫25.2至30.4度，最大平均風力4級；平原區氣溫26.7至30.8度；近山區氣溫25.7至30.6度。

明天(20日)雲林地區受清晨低壓帶及西南風增強影響，為陰短暫陣雨或雷雨的天氣。氣溫方面：沿海區氣溫26至30度；平原區氣溫26至30度；近山區氣溫26至31度。

雲林海面明天(20日)風向偏南風，平均風力5至6雷雨區陣風9級，小浪至中浪，陰局部陣雨或雷雨。從事海邊活動及海上作業，請注意安全。
# 連江縣
陰時多雲短暫陣雨或雷雨，外出請記得攜帶雨具

今日(8月19日)馬祖地區為多雲時陰短暫雨上午有靄的天氣，最高溫29.4度，最低溫26.0度。

明日(8月20日)馬祖地區天氣為陰時多雲短暫陣雨或雷雨，氣溫27至31度，外出請記得攜帶雨具。

海象方面

明日(8月20日)臺灣北部海面及馬祖海面天氣分別為多雲局部陣雨或雷雨及陰時多雲局部陣雨或雷雨，風向皆為西南轉偏南風，風力皆為5至6雷雨區陣風9級下午轉6至7雷雨區陣風10級。
