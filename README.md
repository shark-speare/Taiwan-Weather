昨日(8月8日)馬祖地區天氣為晴時多雲，最高溫32.7度，最低溫26.6度。

今日(8月9日)馬祖地區為多雲時晴的天氣，氣溫27至33度，高溫炎熱，請做好防曬並多補充水分。

海象方面

今日(8月9日)臺灣北部海面及馬祖海面天氣皆為多雲局部陣雨或雷雨，風向分別為西南風及偏西轉西南風，風力分別為4至5雷雨區陣風8級及4級陣風6級以下上午轉4級以下雷雨區陣風8級。
