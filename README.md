今日(10月8日)東北季風影響，馬祖地區為陰天的天氣，氣溫22至23度，氣溫稍下降，請適時增添衣物。易有8至9級強陣風，請注意。

海象方面

今日(10月8日)臺灣北部海面及馬祖海面天氣為陰時多雲局部陣雨，風向分別為東北風及偏北轉東北風，風力分別為6至7陣風9級及5至6陣風8級上午轉6至7陣風9級。
