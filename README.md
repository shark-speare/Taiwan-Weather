明日(9月10日)臺灣北部海面及馬祖海面天氣皆為多雲局部陣雨或雷雨，風向分別為東北轉偏北風及東北風，風力分別為4至5雷雨區陣風8級上午轉5至6雷雨區陣風9級及4至5雷雨區陣風8級。
