# 雲林縣
第１８號颱風及其外圍環流影響，陰短暫陣雨或雷雨。降雨機率80%，外出請備雨具。舒適，請注意做好防颱措施。空曠地區易有較強陣風，沿海有長浪發生的機率，請注意安全。山區有局部大雨或豪雨發生的機率，請注意。

今天(2日)雲林地區受第１８號颱風影響，為陰短暫陣雨或雷雨的天氣。氣溫方面：沿海區氣溫26.2至27.5度，最大平均風力8級；平原區氣溫25.9至28.1度；近山區氣溫25.2至27.6度。

明天(3日)雲林地區受第１８號颱風及其外圍環流影響，為陰短暫陣雨或雷雨的天氣。氣溫方面：沿海區氣溫24至25度；平原區氣溫23至24度；近山區氣溫23至24度。

# 連江縣
