今日(8月18日)低壓帶影響，馬祖地區為多雲的天氣，氣溫27至31度，請做好防曬並多補充水分。

海象方面

今日(8月18日)臺灣北部海面及馬祖海面天氣皆為多雲局部陣雨或雷雨，風向皆為偏西轉西南風，風力分別為4至5雷雨區陣風8級晚轉5至6雷雨區陣風9級及4級以下雷雨區陣風8級晚轉4至5雷雨區陣風8級。
