明日(10月7日)東北季風增強，馬祖地區為陰時多雲短暫陣雨或雷雨的天氣，氣溫23至26度，外出請攜帶雨具備用。將有8至9級強陣風，請注意。

海象方面

明日(10月7日)臺灣北部海面及馬祖海面天氣皆為多雲時陰局部陣雨或雷雨，風向分別為偏東轉東北風及東北風，風力為4至5雷雨區陣風8級上午轉5至6雷雨區陣風9級晚再轉6至7雷雨區陣風10級。
