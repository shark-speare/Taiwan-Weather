今天(18日)雲林地區受低壓帶影響，為多雲時陰的天氣。氣溫方面：沿海區氣溫25.6至31.3度，最大平均風力4級；平原區氣溫26.8至32.1度；近山區氣溫26至31.2度。

明天(19日)雲林地區受低壓帶及西南風增強影響，為陰短暫陣雨或雷雨的天氣。氣溫方面：沿海區氣溫27至29度；平原區氣溫26至28度；近山區氣溫26至28度。

雲林海面明天(19日)風向偏南風，平均風力4至5雷雨區陣風8級下午轉5至6雷雨區陣風9級，小浪轉中浪，陰局部陣雨或雷雨。從事海邊活動及海上作業，請注意安全。
# 連江縣
陰時多雲短暫陣雨或雷雨，天氣炎熱，請多補充水分

今日(8月18日)馬祖地區為多雲時晴的天氣，最高溫30.9度，最低溫25.7度。

明日(8月19日)晨低壓帶影響及西南風增強，馬祖地區為陰時多雲短暫陣雨或雷雨的天氣，氣溫27至31度，外出請記得攜帶雨具。另請注意易有8至9級強陣風。

海象方面

明日(8月19日)臺灣北部海面及馬祖海面天氣為陰或多雲局部陣雨或雷雨，風向皆為西南轉偏南風，風力分別為4至5雷雨區陣風8級晨轉5至6雷雨區陣風9級及4級以下雷雨區陣風8級。
