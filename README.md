今日(9月18日)馬祖地區晴時多雲的天氣，最高溫29.6度，最低溫26.7度。

明日(9月19日)颱風外圍雲系影響，馬祖地區為多雲短暫陣雨或雷雨，氣溫27至31度，外出請攜帶雨具備用，有長浪發生的機率，請注意。

海象方面

明日(9月19日)臺灣北部海面及馬祖海面天氣分別為多雲局部陣雨或雷雨及陰局部陣雨或雷雨，風向皆為東北轉偏南風，風力皆為5至6雷雨區陣風9級上午轉4至5雷雨區陣風8級。
