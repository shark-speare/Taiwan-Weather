昨天(30日)雲林地區受第１８號颱風外圍環流影響，為多雲的天氣。氣溫方面：沿海區氣溫26.8至33.6度，最大平均風力4級；平原區氣溫26.6至33.2度；近山區氣溫26.2至31.6度。

今天(1日)雲林地區受第１８號颱風及其外圍環流影響，為陰時多雲短暫陣雨的天氣。氣溫方面：沿海區氣溫27至31度；平原區氣溫26至32度；近山區氣溫26至32度。

雲林海面今天(1日)風向偏北風，平均風力5至6雷雨區陣風9級上午轉6至7雷雨區陣風10級晚再轉7至8陣風10級，大浪，多雲局部陣雨或雷雨。從事海邊活動及海上作業，請注意安全。
# 連江縣
多雲短暫陣雨或雷雨，請注意強陣風以及長浪發生。

昨日(9月30日)馬祖地區為陰時多雲上午有短暫雨的天氣，最高溫27.8度，最低溫25.5度。

今日(10月1日)颱風外圍環流影響，馬祖地區天氣為多雲時陰陣雨或雷雨，氣溫24至27度，請注意強陣風以及長浪發生。

海象方面

今日(10月1日)臺灣北部海面及馬祖海面天氣皆為多雲時陰局部陣雨或雷雨，風向分別為偏東轉東北風及東北風，風力分別為6至7雷雨區陣風10級上午轉7至8陣風10級晚再轉8至9陣風11級及6至7雷雨區陣風10級下午轉7至8陣風10級。
