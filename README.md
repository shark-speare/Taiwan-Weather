昨天(20日)雲林地區受南方雲系北移影響，為多雲午後短暫雷陣雨的天氣。氣溫方面：沿海區氣溫25.6至33.6度，最大平均風力3級；平原區氣溫25.9至32.8度；近山區氣溫24.6至32.8度。

今天(21日)雲林地區受熱帶性低氣壓外圍雲系及低壓帶影響，為陰時多雲短暫陣雨或雷雨的天氣。氣溫方面：沿海區氣溫26至32度；平原區氣溫26至32度；近山區氣溫26至32度。

雲林海面今天(21日)風向東南轉偏南風，平均風力4級以下雷雨區陣風8級，中浪轉小浪，陰時多雲局部陣雨或雷雨。從事海邊活動及海上作業，請注意安全。
# 連江縣
多雲短暫陣雨或雷雨，有長浪發生的機率，請注意。

昨日(9月20日)馬祖地區為晴時多雲的天氣，最高溫31.9度，最低溫26.6度。

今日(9月21日)熱帶性低氣壓外圍雲系及低壓帶影響，馬祖地區為多雲短暫陣雨或雷雨，氣溫26至31度，外出請攜帶雨具備用，有長浪發生的機率，請注意。

海象方面

今日(9月21日)臺灣北部海面及馬祖海面天氣分別為多雲局部陣雨或雷雨及陰局部陣雨或雷雨，風向分別為偏北風及西北轉偏北風，風力分別為4至5雷雨區陣風8級晚轉5至6雷雨區陣風9級及4級以下雷雨區陣風8級晚轉5至6雷雨區陣風9級。
