明天(2日)雲林地區受第１８號颱風影響，為陰短暫陣雨或雷雨的天氣。氣溫方面：沿海區氣溫27至30度；平原區氣溫26至31度；近山區氣溫26至31度。

# 連江縣
陰短暫陣雨或雷雨，請注意將有12級以上強陣風以及長浪發生。

今日(10月1日)馬祖地區為陰時多雲有短暫雨的天氣，最高溫27.8度，最低溫24.6度。

明日(10月2日)颱風影響，馬祖地區天氣為陰短暫陣雨或雷雨，天氣轉涼，氣溫21至25度，風力逐漸增強，將有12級以上強陣風以及長浪發生，請注意。
