今天(23日)雲林地區受鋒面通過臺灣北部海面及低壓帶影響，為多雲時陰局部地區短暫陣雨或雷雨的天氣。氣溫方面：沿海區氣溫25.5至32.3度，最大平均風力4級；平原區氣溫24.8至33.7度；近山區氣溫24.8至32.9度。

明天(24日)雲林地區天氣陰時多雲。氣溫方面：沿海區氣溫26至31度；平原區氣溫26至32度；近山區氣溫25至32度。

雲林海面明天(24日)風向偏南轉西南風，平均風力4至5雷雨區陣風8級，中浪轉小浪，陰局部陣雨或雷雨。從事海邊活動及海上作業，請注意安全。
# 連江縣
陰短暫陣雨或雷雨，外出請攜帶雨具備用，易有8至9級強陣風，請注意。

昨日(9月22日)馬祖地區為陰有雨的天氣，最高溫27.6度，最低溫22.6度。

今日(9月23日)清晨鋒面及低壓帶影響，馬祖地區為陰短暫陣雨或雷雨，氣溫22至26度， 外出請攜帶雨具備用。易有8至9級強陣風，請注意。

海象方面

今日(9月23日)臺灣北部海面及馬祖海面天氣皆為陰局部陣雨或雷雨，風向皆為東北轉偏北風，風力皆為7至8陣風10級上午轉6至7雷雨區陣風10級下午再轉5至6雷雨區陣風9級。
