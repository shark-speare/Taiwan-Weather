# 連江縣
多雲短暫陣雨或雷雨，請注意強陣風。

今日(9月29日)馬祖地區為陰時多雲上午有短暫毛雨的天氣，最高溫27.8度，最低溫23.5度。

明日(9月30日)颱風外圍環流影響，馬祖地區天氣為多雲短暫陣雨或雷雨，氣溫26至28度，外出請攜帶雨具備用並請注意8至9級強陣風。

海象方面
