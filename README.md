今天(21日)雲林地區受熱帶性低氣壓外圍雲系及低壓帶影響，為陰時多雲短暫陣雨或雷雨的天氣。氣溫方面：沿海區氣溫26.3至29.7度，最大平均風力4級；平原區氣溫26.2至29.5度；近山區氣溫25.6至29.7度。

明天(22日)雲林地區受低壓帶影響，為陰短暫陣雨或雷雨的天氣。氣溫方面：沿海區氣溫26至29度；平原區氣溫26至28度；近山區氣溫26至28度。

雲林海面明天(22日)風向偏南風，平均風力4級以下雷雨區陣風8級晨轉5至6雷雨區陣風9級，小浪轉中浪再轉大浪，陰局部陣雨或雷雨。從事海邊活動及海上作業，請注意安全。
# 連江縣
陰陣雨或雷雨，有長浪發生的機率，請注意。

今日(9月21日)馬祖地區為晴時多雲的天氣，最高溫31.2度，最低溫26.7度。

明日(9月22日)低壓帶影響，馬祖地區為陰陣雨或雷雨，氣溫24至27度， 外出請攜帶雨具備用，有長浪發生的機率，請注意。

海象方面

明日(9月22日)臺灣北部海面及馬祖海面天氣皆為陰局部陣雨或雷雨，風向皆為東北轉偏北風，風力皆為7至8陣風10級。
