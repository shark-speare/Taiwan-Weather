明日(9月26日)鋒面通過，馬祖地區天氣為陰時多雲短暫陣雨或雷雨，氣溫25至28度，外出請攜帶雨具備用，白天降雨緩和，氣溫稍回升。

海象方面

明日(9月26日)臺灣北部海面及馬祖海面天氣為多雲或陰時多雲有局部陣雨或雷雨，風向分別為東北風及偏東轉東北風，風力分別為4至5雷雨區陣風8級晨轉4級以下雷雨區陣風8級下午再轉4至5雷雨區陣風8級及4級以下雷雨區陣風8級。
