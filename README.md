第１８號颱風影響，陰短暫陣雨或雷雨。降雨機率50%，外出請備雨具。舒適至悶熱，請注意防曬及適時補充水分。有局部短暫陣雨或雷雨，山區有局部大雨或豪雨，請注意。

昨天(1日)雲林地區受第１８號颱風及其外圍環流影響，為陰時多雲短暫陣雨的天氣。氣溫方面：沿海區氣溫27.8至31.3度，最大平均風力3級；平原區氣溫27.6至31.1度；近山區氣溫26.5至30.2度。

今天(2日)雲林地區受第１８號颱風影響，為陰短暫陣雨或雷雨的天氣。氣溫方面：沿海區氣溫27至30度；平原區氣溫26至31度；近山區氣溫26至31度。

雲林海面今天(2日)風向風，平均風力，，。從事海邊活動及海上作業，請注意安全。
# 連江縣
陰短暫陣雨或雷雨，請注意將有11級強陣風以及長浪發生。

昨日(10月1日)馬祖地區為陰時多雲有短暫雨的天氣，最高溫27.8度，最低溫24.6度。

今日(10月2日)颱風影響，馬祖地區天氣為陰短暫陣雨或雷雨，天氣轉涼，氣溫21至24度，風力逐漸增強，將有11級強陣風以及長浪發生，請注意。

海象方面

今日(10月2日)臺灣北部海面及馬祖海面天氣皆為陰局部陣雨或雷雨，風向皆為偏北轉東北風，風力分別為8至9陣風11級及7至8陣風10級上午轉8至9陣風11級。
