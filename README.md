雲林海面明天(20日)風向偏南轉東南風，平均風力4級以下雷雨區陣風8級晨轉4至5雷雨區陣風8級，小浪轉中浪，多雲時陰局部陣雨或雷雨。從事海邊活動及海上作業，請注意安全。
