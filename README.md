低壓帶影響，多雲時陰局部地區短暫陣雨。降雨機率20-30%，外出請備雨具。舒適至悶熱，請注意防曬及適時補充水分。易有短延時強降雨，午後並有局部大雨或豪雨發生的機率，請注意。
