昨天(3日)雲林地區受第１８號颱風及其外圍環流影響，為陰短暫陣雨或雷雨的天氣。氣溫方面：沿海區氣溫25.5至26.2度，最大平均風力7級；平原區氣溫24.5至26.3度；近山區氣溫23.8至26.3度。

今天(4日)雲林地區受颱風或熱帶性低氣壓及其外圍環流影響，為陰時多雲短暫陣雨的天氣。氣溫方面：沿海區氣溫24至28度；平原區氣溫23至28度；近山區氣溫23至28度。

雲林海面今天(4日)風向偏北轉東北風，平均風力7至8陣風10級，巨浪轉大浪，陰局部陣雨或雷雨。從事海邊活動及海上作業，請注意安全。
# 連江縣
陰短暫陣雨或雷雨，請注意易有長浪發生。

昨日(10月3日)馬祖地區為陰有短暫雨的天氣，溫度下降，最高溫20.9度，最低溫19.9度。

今日(10月4日)颱風或熱帶性低氣壓及其外圍環流影響，馬祖地區為陰短暫陣雨或雷雨，氣溫19至23度，外出請攜帶雨具備用，請長浪發生。

海象方面

今日(10月4日)臺灣北部海面及馬祖海面天氣為陰局部陣雨或雷雨，風向分別為東北轉東南風及偏北風，風力分別為 8至9陣風11級上午轉7至8陣風10級及7至8陣風10級。
