今天(28日)雲林地區受東北風增強及熱帶性低氣壓或颱風外圍雲系影響，為多雲時晴的天氣。氣溫方面：沿海區氣溫26.4至31.4度，最大平均風力5級；平原區氣溫25.6至31.9度；近山區氣溫25.4至32.4度。

明天(29日)雲林地區受東北風及第１８號颱風外圍雲系影響，為多雲短暫陣雨或雷雨的天氣。氣溫方面：沿海區氣溫26至32度；平原區氣溫26至33度；近山區氣溫25至33度。

雲林海面明天(29日)風向東北風，平均風力6至7陣風9級，中浪至大浪，多雲時晴。從事海邊活動及海上作業，請注意安全。
# 連江縣
多雲短暫陣雨或雷雨，外出請攜帶雨具備用。

今日(9月28日)馬祖地區為多雲時晴的天氣，最高溫27.2度，最低溫24.6度。

明日(9月29日)東北風及颱風外圍雲系影響，馬祖地區為陰時多雲短暫陣雨或雷雨，氣溫25至28度，外出請攜帶雨具備用。

海象方面

明日(9月29日)臺灣北部海面及馬祖海面天氣皆為多雲局部陣雨或雷雨，風向分別為為東北轉偏東風及偏北轉東北風，風力為6至7雷雨區陣風10級及5至6雷雨區陣風9級晨轉6至7雷雨區陣風10級。
