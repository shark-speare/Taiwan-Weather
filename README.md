今天(22日)彰化地區受低壓帶影響，為陰短暫陣雨或雷雨的天氣。氣溫方面：濱海區氣溫24.9至28.9度，最大平均風力4級；西平原區氣溫27.2至31.2度；東平原區氣溫25.4至28.9度。

明天(23日)彰化地區受鋒面通過臺灣北部海面及低壓帶影響，為陰時多雲短暫陣雨或雷雨的天氣。氣溫方面：濱海區氣溫25至29度；西平原區氣溫25至29度；東平原區氣溫25至30度。

彰化海面明天(23日)風向東北轉偏南風，平均風力6至7雷雨區陣風10級上午轉4至5雷雨區陣風8級，大浪轉中浪，陰局部陣雨或雷雨。從事海邊活動及海上作業，請注意安全。
# 雲林縣
鋒面通過臺灣北部海面及低壓帶影響，多雲時陰局部地區短暫陣雨或雷雨。降雨機率20-50%，外出請備雨具。舒適至悶熱，請注意適時補充水分。有局部較大雨勢發生的機率，請注意。

今天(22日)雲林地區受低壓帶影響，為陰短暫陣雨或雷雨的天氣。氣溫方面：沿海區氣溫24.6至28.3度，最大平均風力4級；平原區氣溫24.7至27.3度；近山區氣溫24.5至27.6度。

明天(23日)雲林地區受鋒面通過臺灣北部海面及低壓帶影響，為多雲時陰局部地區短暫陣雨或雷雨的天氣。氣溫方面：沿海區氣溫25至29度；平原區氣溫24至29度；近山區氣溫24至29度。

雲林海面明天(23日)風向偏南風，平均風力5至6雷雨區陣風9級上午轉4至5雷雨區陣風8級，中浪至大浪，陰局部陣雨或雷雨。從事海邊活動及海上作業，請注意安全。
# 連江縣
陰短暫陣雨或雷雨，外出請攜帶雨具備用。

今日(9月22日)馬祖地區為陰有雨的天氣，最高溫27.6度，最低溫22.6度。

明日(9月23日)清晨鋒面及低壓帶影響，馬祖地區為陰短暫陣雨或雷雨，氣溫22至26度， 外出請攜帶雨具備用。

海象方面

明日(9月23日)臺灣北部海面及馬祖海面天氣皆為陰局部陣雨或雷雨，風向分別為東北轉西南風及東北轉偏南風，風力皆為7至8陣風10級下午轉6至7雷雨區陣風10級。
