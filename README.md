明日(8月9日)臺灣北部海面及馬祖海面天氣皆為多雲局部陣雨或雷雨，風向分別為西南風及偏南轉西南風，風力為4級以下雷雨區陣風8級，臺灣北部海面晨轉4至5雷雨區陣風8級。
