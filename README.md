今天(3日)彰化地區受第１８號颱風及其外圍環流影響，為陰短暫陣雨或雷雨的天氣。氣溫方面：濱海區氣溫24.6至25.9度，最大平均風力6級；西平原區氣溫25至26.7度；東平原區氣溫25.3至26.7度。

明天(4日)彰化地區受颱風或熱帶性低氣壓及其外圍環流影響，為陰時多雲短暫陣雨的天氣。氣溫方面：濱海區氣溫24至28度；西平原區氣溫24至28度；東平原區氣溫23至29度。

彰化海面明天(4日)風向東北轉偏北風，平均風力9至10陣風12級晨轉8至9陣風11級上午再轉7至8陣風10級，猛浪轉巨浪，陰時多雲局部陣雨或雷雨。從事海邊活動及海上作業，請注意安全。

# 連江縣
陰短暫陣雨或雷雨，請注意易有11級以上強陣風及長浪發生。

今日(10月3日)馬祖地區為陰有短暫雨的天氣，溫度下降，最高溫20.9度，最低溫19.9度。今晚易有11級以上強陣風，請注意。

明日(10月4日)颱風或熱帶性低氣壓及其外圍環流影響，馬祖地區為陰短暫陣雨或雷雨，氣溫20至23度，請注意強陣風及長浪發生。

海象方面

明日(10月4日)臺灣北部海面及馬祖海面天氣為陰局部陣雨或雷雨，風向分別為東北轉東南風及偏北風，風力分別為9至10陣風12級晨轉8至9陣風11級上午再轉7至8陣風10級及8至9陣風11級上午轉7至8陣風10級。
